# MissBayes-DataAnalysis
Code of figures and results for the manuscript: Empirical-Bayes and Bayesian Hierarchical Modelling for Missingness and Differential Expression in Proteomics.
Both CQE-A and CQE-B analyses use the same pipeline implemented in CQE_data_analysis.R. For CQE-B, only the comparison groups were modified accordingly, while the remaining analysis workflow remained unchanged.
